# 学校心理预约小程序：视觉对标与功能规划

## 调研目标

为本校学生端确定一套长期稳定的视觉系统，并把后续新增功能限定在“预约服务、安全支持和持续关怀”范围内，避免把心理预约小程序做成内容商城或复杂医疗平台。

## 对标产品与可借鉴模式

### 1. 微信 WeUI：原生、可预期、低学习成本

- 微信官方 WeUI 由微信设计团队和小程序团队提供，目标是让小程序与微信原生体验保持一致。
- 借鉴：按钮、表单、弹窗、列表、状态反馈优先遵循微信习惯；减少自造控件。
- 不照搬：WeUI 只解决基础交互，不能替代学校心理服务自身的品牌和信任表达。
- 来源：https://wechat-miniprogram.github.io/weui/docs/

### 2. 京东健康 / 好大夫在线：身份可信和任务效率

- 京东健康通过医生资质、科室、接诊状态、隐私保障和明确服务边界建立医疗信任。
- 好大夫在线把“找医生、看详情、预约、订单状态”组织为明确任务链。
- 借鉴：咨询师真实头像、职称/受训背景、校内认证、最近可约时间、地点和预约凭证必须清晰。
- 不照搬：不引入价格、销量、促销、评价排行和医疗诊断语气。
- 来源：https://m.jd.com/health/
- 来源：https://www.haodf.com/

### 3. 简单心理：先评估、再匹配、全周期支持

- 简单心理强调心理评估、服务匹配、咨询和持续支持，并把专业、私密、温暖作为信任支柱。
- 借鉴：首次预约引导、咨询师匹配、咨询前准备、咨询后的回访与持续支持。
- 不照搬：学校场景不需要商业课程、付费内容和复杂社区。
- 来源：https://www.jiandanxinli.com/
- 来源：https://apps.apple.com/cn/app/id951454292

### 4. 壹心理：咨询师筛选和专业信息密度

- 壹心理的咨询师列表支持按困扰类型、咨询方式等筛选，并呈现从业经历和专业背景。
- 借鉴：学生可按学业压力、情绪、人际、生涯等主题筛选；详情页应展示学校确认过的专业信息。
- 不照搬：学校内部服务不使用咨询次数、价格和商业排序。
- 来源：https://www.xinli001.com/consult

### 5. Headspace：情绪安全和单一主任务

- Headspace 用明确的单一行动、友好插画和短内容降低心理健康产品的进入压力。
- 借鉴：首页只突出“管理当前预约”或“开始预约”一个主要任务；插画只承担情绪缓冲，不占据信息区域。
- 不照搬：不把预约系统扩展成冥想内容库，也不使用大面积娱乐化动画。
- 来源：https://www.headspace.com/app

## 三个待选视觉方向

### 医疗级校务

- 关键词：正式、可靠、克制、快速扫描。
- 结构：白底、深墨绿/医疗蓝、轻分隔线、真实头像、时间表优先。
- 适合：学校希望突出官方身份、流程与安全责任。

### 温暖专业

- 关键词：陪伴、安静、人本、不过度医疗化。
- 结构：柔和中性色、局部珊瑚色、真实人物与少量授权插画、内容留白更大。
- 适合：希望降低学生求助心理门槛，同时保持专业可信。

### 青年校园

- 关键词：轻快、清晰、校园感、低压力。
- 结构：清爽浅色、校园摄影或授权图形、时间卡与快捷入口更活跃，但避免游戏化积分。
- 适合：希望提高学生使用意愿，并承载活动、资源与轻量自助功能。

三套方向都必须保留：微信原生交互、风险红色语义、44px 级触控目标、清晰中文层级、无商业促销、无娱乐化危机提示。

## 推荐新增功能

### 第一批：与预约闭环直接相关

1. 微信订阅消息：预约成功、变更、取消截止和到访前提醒。
2. 候补排队：时段满额后允许加入候补，释放时按规则通知。
3. 快速复约：从已完成或已取消记录重新选择同一咨询师时段。
4. 到访指引：校区、房间、地图或楼层文字说明。
5. 预约后反馈：仅收集流程体验、是否解决预约问题，不公开给其他学生。
6. 咨询前准备：首次咨询须知、需要携带的信息、提前到达提醒。

### 第二批：学校心理服务常见扩展

1. 团体辅导和心理活动预约。
2. 心理健康资源：学校审核的文章、音频和常见问题。
3. 情绪记录：私密、可主动删除，不默认共享给咨询师或管理端。
4. 专业量表自测：必须使用经授权、适合学校场景的量表，结果附边界说明和求助建议。
5. 咨询后回访：由管理端配置时间和问题，支持再次预约或转介。
6. 服务评价与申诉：评价不公开，管理员可处理服务问题。

### 暂不建议

1. 匿名树洞、公开社区、评论区和私信：内容治理、危机识别和校园舆情风险过高。
2. AI 心理治疗或自动诊断：当前项目应把 AI 限定为导航、FAQ 或资源检索，不能替代专业咨询。
3. 积分、连续签到和排名：容易把求助行为游戏化，并产生不适当激励。
4. 咨询师公开评分和销量排行：会伤害学校资源公平分配，也可能误导学生判断专业适配度。

## 风格冻结规则

用户选择最终方向后，建立一份不可随意变更的设计系统，至少冻结：

- 品牌色、功能色、风险色和中性色阶。
- 字体、字号、字重和行高。
- 页面边距、间距梯度、圆角和分隔线。
- 按钮、输入框、标签、列表、弹窗、状态和空页面组件。
- 图标库、头像规则、插画/摄影规则。
- 学生端、咨询师端和后台管理端之间的共用语义与差异。

后续只有业务需求变化、无障碍问题或真实设备兼容问题可以触发设计系统变更，并需要记录变更原因。
